# Instructions Supported
- Data Transfer Group

  `-` MOV, MVI, LDA, LDAX, STA, STAX,    XCHG, XTHL, SPHL, PCHL, LHLD, SHLD

- Arithmetic Group

  `-` ADD, ADI, ADC, ACI, SUB, SUI, SBB, SBI, INR, DCR, DAD, INX, DCX, DAA

- Logical Group

  `-` AND, ANI, ORA, ORI, XRA, XRI, CMP, CPI, CMA, RLC, RRC, RAL, RAR

- Branch/Jump Instructions

  `-` JMP, JC, JNC, JZ, JNZ, JP, JM, JPE, JPO, PCHL

- Flag Manipulations

   `-` STC, CMC

- Stack Instructions

   `-` PUSH, POP

- I/O Instructions

   `-` IN, OUT

- Machine Control Instruction
   
   `-` HLT
